# Boards and Views

TaskFlow lets you view the same project in four different ways depending on
what you're trying to do. All views show the same underlying tasks — changes
in one view appear instantly in all the others.

## Kanban view

The default view for most projects. Tasks appear as cards in columns, where
each column represents a status (To Do, In Progress, In Review, Done).

**Best for:** tracking work in progress and spotting bottlenecks at a glance.

You can drag cards between columns to update their status, and you can
customize column names from **Project settings → Columns**.

## List view

A compact, spreadsheet-like table of all tasks in the project. Every task
property (assignee, due date, priority, labels) appears as a sortable column.

**Best for:** bulk editing, sorting by due date, or reviewing large projects.

You can click any column header to sort, and use the filter bar at the top
to narrow down tasks by assignee, label, or status.

## Calendar view

Tasks appear on a monthly calendar based on their due dates. Tasks without a
due date are listed in a sidebar so you can drag them onto the calendar.

**Best for:** planning around deadlines and spotting days with too much work.

You can switch between **month**, **week**, and **day** zoom levels.

## Timeline view (Gantt)

A horizontal timeline showing tasks as bars spanning their start and due
dates. You can create dependencies between tasks by dragging from the edge of
one bar to another.

**Best for:** long-running projects with dependencies and milestones.

Timeline view is available on the **Pro**, **Business**, and **Enterprise**
plans only. Free users see a locked preview with an upgrade prompt.

## Switching views

You can switch views at any time using the view selector in the top-right
corner of any project. TaskFlow remembers the view you last used for each
project, so the next time you open it you'll land back where you left off.

## Saving custom views

On Business and Enterprise plans, you can save filtered versions of any view
as a **custom view**. For example, you could save "My overdue tasks" or
"Design team — this week". Custom views appear in the sidebar for quick
access.

## Frequently asked questions

**Can I change the default view for a project?**
Yes. Go to **Project settings → General → Default view** and pick one. New
members joining the project will see this view first.

**Do filters I apply affect what my teammates see?**
No. Filters are personal to you. If you filter the Kanban view to show only
tasks assigned to you, your teammates still see everything.

**Can I export a view?**
Yes. From any view, click **··· → Export** to download the visible tasks as
CSV. See the export data guide for full details.
