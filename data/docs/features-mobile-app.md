# Mobile Apps

TaskFlow offers native mobile apps for iOS and Android so you can stay on
top of your work from anywhere. Both apps are free to download and work
with any TaskFlow plan.

## Download

- **iOS:** available on the App Store, requires iOS 15 or later
- **Android:** available on Google Play, requires Android 9 (Pie) or later

The same login credentials you use on the web work on mobile — no separate
account is needed.

## What works on mobile

Most TaskFlow features are fully supported on mobile:

- **Viewing projects** in Kanban and List views
- **Creating, editing, and completing tasks**
- **Adding comments and @mentions**
- **Uploading attachments** (including photos taken from the camera)
- **Push notifications** for mentions, assignments, and due dates
- **Offline mode** — you can view and edit tasks without an internet
  connection, and changes sync automatically when you reconnect

## What is web-only

A few features are only available in the web app:

- **Timeline (Gantt) view** — the UI is designed for larger screens
- **Workspace and billing settings**
- **Bulk CSV import**
- **Integration setup** (you can use integrations on mobile once they're
  connected, but initial configuration must be done on the web)
- **Custom view creation** (you can view saved custom views on mobile, but
  you must create them on the web)

## Offline mode

When you lose connection, TaskFlow switches to offline mode automatically.
You'll see a small indicator at the top of the screen. While offline:

- You can browse tasks and projects you've already opened
- You can create new tasks, add comments, and change statuses
- Changes are queued locally and sync when you reconnect
- If two people edit the same task while offline, the most recent change
  wins and the other is discarded with a warning

## Widgets

iOS and Android both support home-screen widgets:

- **Today's tasks** — a list of your tasks due today
- **Quick add** — a shortcut that opens the task creation screen
- **Project progress** — a pie chart of one project's completion status

## Push notifications

Mobile apps support push notifications for:

- @mentions in comments
- New task assignments
- Upcoming due dates (1 hour before, 1 day before, configurable)
- Task status changes on tasks you're watching

You can customize which notifications you receive from
**Settings → Notifications → Mobile** in either the mobile or web app.

## Frequently asked questions

**Can I use TaskFlow on an iPad?**
Yes. The iOS app runs on iPad and adapts its layout to the larger screen,
showing a sidebar alongside the main view.

**Does the mobile app support biometric login?**
Yes. You can enable Face ID, Touch ID, or fingerprint unlock from
**Settings → Security** in the mobile app.

**How much data does the mobile app use?**
The app is designed to be lightweight. A typical user under normal use
consumes less than 10 MB of data per day. Attachments count separately.

**Why can't I see the Timeline view on my phone?**
Timeline (Gantt) view is web-only because it requires a larger screen to be
usable. You can still see tasks with start and due dates in the Calendar
view, which is available on mobile.
